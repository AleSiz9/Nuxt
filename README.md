**🚀 Установка и запуск.**
Клонируйте репозиторий (или создайте проект по инструкции). 

##Установите зависимости: 
`npm install`
##Запустите dev-сервер:
`npm run dev`

#🛒 Nuxt 3 Pet-проект: Корзина бытовой техники.
Это pet-проект, созданный для обучения. Предложения и улучшения приветствуются. 

**✨ Возможности.**
🔍 Просмотренные товары — карусель с адаптивной пагинацией (кнопки «← / →» и счётчик страниц). 

🛍️ Корзина: 
Добавление/удаление товаров, изменение количества. 

Опция «Установка» с чекбоксом. 

Подсчёт общей суммы и количества. 

Очистка корзины. 

✅ Оформление заказа: 

Отправка данных на сервер Nuxt (Nitro). 

Сохранение заказов в файл server/data/orders.json. 

🧩 Кастомная карусель на scroll-snap вместо сторонних библиотек. 

**🛠️ Технологии:**
Nuxt 3 (Vue 3, Vite, Nitro). 
Pinia — управление состоянием. 
TypeScript. 
Node.js (встроенный сервер Nitro).
